---

## Установка через npm 📦

```bash
npm install IrisAPIClientTS --version 2.0.0
```
---

## Как получить токен?

Для начала вам нужно создать бота в [BotFather](https://t.me/BotFather), далее в [Iris | Black Diamond](https://t.me/iris_cm) написать: `+ирис коннект` и следовать инструкции.
